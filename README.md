# Tech Path | مسار التقنية

A bilingual beginner-friendly learning platform built as a static website for GitHub Pages.

## Features
- Arabic / English
- Light / Dark mode
- 7 complete color themes
- Ordered learning path: SRS → Database → SQL/NoSQL → ERD → Schema → SQL → APIs → AI → Career tools
- Complete SQL reference for DDL, DML, DCL, objects, joins, grouping, and TCL
- SQL examples and SQL-only quiz
- Progress saved in LocalStorage
- Fully responsive and no backend required

## GitHub Pages
Upload all files to the repository root, then enable:
`Settings → Pages → Deploy from a branch → main → /root`
